# Database Schema

All entities are generated by `sea-orm-codegen` and live in `src/entities/`. Every table uses a UUID primary key (non-auto-increment) and includes `created_at` / `updated_at` timestamps unless noted.

---

## Entity Relationship Overview

```
accounts
  |-- projects        (has_many, via created_by)
  |-- tasks           (has_many, via created_by)
  |-- answers         (has_many, via user_id)
  |-- attachments     (has_many, via created_by)
  |-- moderators      (has_many, via user_id)

projects
  |-- tasks           (has_many, cascade delete)
  |-- moderators      (has_many, cascade delete)
  |-- participants    (has_many, cascade delete)

tasks
  |-- questions       (has_many, cascade delete)
  |-- attachments     (has_many, cascade delete)

questions
  |-- answers         (has_many, cascade delete)
  |-- attachments     (has_many, cascade delete)

answers
  |-- attachments     (has_many, cascade delete)
```
